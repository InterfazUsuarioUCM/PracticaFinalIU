Decisiones de diseño

Durante el desarrollo de esta práctica hemos utilizado varios de los patrones que aprendimos durante el año en la asignatura. En este documento explicamos qué patrones sí hemos usado y cuáles hemos descartado a la hora de realizar la remodelación de la página web de la OPE.


Patrones de estética

Patrones utilizados

Paleta de colores

Para el rediseño de la página web de la OPE, hemos elegido una paleta coherente con los colores identificativos de la UCM, por lo que utilizamos los principales colores del escudo de la Universidad como dicta el manual de la Marca UCM, para que el usuario pueda identificar fácilmente la procedencia de la web.

Deep Background

Con el uso del Deep Background, intentamos conseguir una imagen corporativa parecida a la de UCM, que utiliza este patrón en su página principal. Además, al tener una página principal despejada, el empleo de un Deep Background no abruma al usuario, ni desvía su atención de elementos más importantes.

Corner Treatments

Para conseguir que haya una consistencia entre todos los elementos de la página web, hemos utilizado bordes cuadrados para los componentes principales que la forman, así, siguiendo el patrón Coerner Treatments, logramos unificar el diseño que tiene nuestra página.

Border That Echo Fonts

Para intentar alcanzar la consistencia hablada en el punto anterior, hemos hecho uso el patrón Border That Echo Fonts, que nos permite mantener una coherencia entre el tipo de fuente que utilizamos y la forma de los elementos principales de la página.

Contrasting Font Weights

El uso de diferentes fuentes, nos permite destacar diferentes niveles de información y hacer que el usuario centre su atención en elementos más significativos dentro de los contenidos de la página.


Patrones de navegación

Patrones utilizados

Escape Hatch

Para facilitar al usuario la navegación por la página web, hemos utilizado el banner principal como punto de escape, para permitir al usuario volver a la página principal de una manera sencilla, y así, darle confianza para explorar la interfaz sin sentirse atrapado en ningún momento, tal y como indica el patrón Escape Hatch.

Breadcumbs

Hemos hecho uso de Breadcumbs en el diseño final de la página de la OPE, ya que al tener una estructura jerárquica, nos permite conseguir que el usuario sepa donde en cualquier momento con mayor facilidad.

Animated Transition

Con el fin de hacer la página web más atractiva para el usuario y darle una mayor sensación de continuidad a los contenidos, hemos hecho uso de diferentes transiciones a la hora hacer aparecer y desaparecer elementos de esta.

Patrones descartados

Fat Menu

Hemos descartado el uso de un Fat Menu en nuestra página web porque no teníamos un número de elementos suficiente, ni una profundidad tan elevada como para tener que hacerlo, por lo que nos hemos decantado por un menú sencillo en el que se muestras las categorías disponibles y los contenidos que hay en ellas.


Patrones de disposición del contenido

Patrones utilizados

Grid of Equals

Hemos utilizado un Grid of Equals para distribuir el contenido de nuestras secciones, ya que nos ofrece una manera limpia y sencilla de mostrar al usuario todas las opciones que puede realizar en la página web, sin abrumarle con un exceso de información, como ocurre en la página web actual de la OPE.
