Memoria Grupo 10

Kingdom hearts 3

Esta memoria reune todo el proceso realizado para el Proyecto Final de Interfaces de Usuario. Hemos elegido analizar diferentes páginas de videojuegos indies que han tenido un éxito reciente para llegar a unas conclusiones finales de como debería ser una pagina de presentación de un videojuego y la elaboración de un ejemplo. Para la elaboración del ejemplo hemos escogido una licencia conocida, Kingdom Hearts 3, para no tener complicaciones con el arte y hemos desarrollado la página de presentación del videojuego.
